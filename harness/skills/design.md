# Skill: UI Design

## Use When

Use for session workflow design, component composition, layout, interaction states, and persona debate presentation.

## Steps

1. Read `harness/agents/designer.md`.
2. Check current BDD scenarios.
3. Design the usable product screen first, not a landing page.
4. Keep writing, AI response, and saved history states visually distinct.
5. Check mobile text wrapping and control density.

## Done

- User can read, write, and review without visual noise.
- Persona identity and AI streaming states are understandable.
- Layout choices do not break E2E selector strategy.
