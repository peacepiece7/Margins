# Owner 결정 계획 기록

## 페이지 구성 수정

- 로그인 페이지
- 책 검색 및 등록 페이지
- 등록 책 리스트 페이지
- 등록 책 상세 페이지
- 독후감 페이지
- 토론 세션 페이지

## 로그인 페이지

기존 로그인 페이지와 동일

## 책 검색 및 등록 페이지

책 검색 -> 도서 등록 가능해야함
외부 도서 검색 API로 책의 고유번호, 책이름, 저자 등을 조회 가능
만약 없을 경우 사용자가 임의로 책이름, 책 저자 등 추가해서 책 등록이 가능

## 등록 책 리스트 페이지

책 검색 및 등록을 완료한 리스트가 나옴
리스트는 클릭 가능하고 클릭시 책 상세 페이지로 이동한다.

수정, 삭제 기능도 제공

## 등록 책 상세 페이지

### 독후감 입력 기능
클릭시 해당 책에 대한 독후감 작성가능 독후감 페이지로 이동함 여기에는 꼭 독후감이 아니라도 좋았던 점 등 개인 의견을 기록하는 공간임

### 질문지 리스트

해당 책에 대해서 좋았던점이나 어떤 구절이 좋았는지 등등 각종 질문이 있고 거기에 대해서 답변할 수 있음.
질문지는 OPEN AI를 사용해서 해당 책, 저자, 책고유번호를 주고 어떤 질문이 좋을지 리스트를 뽑아서 클라이언트에 제공함
각질문에 대해서 클릭시 독후감 작성 페이지로 이동하고 질문에 대해서 답변 작성가능 

### 토론하기 기능

해당 기능을 클릭하면 토론 세션에 진입함
여기서는 3명의 페르소나와 나 이렇게 총 4명이서 대화하는 토론 세션에 진입하게됨
메시지를 주고 받는 것 같은 세션으로 구성 되어있고 AI는 각각 이 책에 대해서 토론을 이어감.
하나의 주제에 대해서 깊게 토론하는 세션으로 사용자가 토론 세션에 진입하기전에 어떤 주제에 대해서 토론하고싶은지 미리 주제를 정해야함
